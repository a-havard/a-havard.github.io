Expand description

Something that can be seen as an immutable slice

NOTE: This trait is implemented for arrays ([T; N]) of sizes 0 to 256 (inclusive) and arrays whose lengths are a power of 2 up to 1 << 16. These implementations don’t show in the documentation because they would reduce readability.
