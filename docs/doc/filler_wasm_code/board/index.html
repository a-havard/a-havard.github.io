Module filler_wasm_code::board

source · [−]
Expand description

Describes a board object which contains both an 8x7 board, as well as the current player whose turn it is to move. This file also includes utility functions to convert board cells from chars to ints and vice versa.

Here is the relationship between colors, chars, and the corresponding int.

ColorCharInt
RedR0
GreenG1
YellowY2
BlueU3
PurpleP4
BlackK5

The following functions are exposed to Javascript:

  • from_str()
  • get_player_turn()
  • get_string()
  • get_cell()
  • run_turn()
  • get_player_squares()
