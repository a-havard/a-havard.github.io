pub fn new(size: usize, active: bool) -> DistributionTimer

Distribution Timer constructor, technically the size is a parameter but as of now, the size is hard-coded whenever you taken in labels. Maybe that will be fixed, maybe it won’t.

If you are running a release version of the code, set active to false to minimize the impact of the timer on the code. No calculations will be run.
